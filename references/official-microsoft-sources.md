# Official Microsoft Sources & Wording

## Azure Hybrid Benefit
Official page: https://azure.microsoft.com/pricing/hybrid-benefit/
Official description: "Azure Hybrid Benefit is a licensing benefit that
helps you to significantly reduce the costs of running your workloads
in the cloud. It works by letting you use your on-premises
Software Assurance-enabled Windows Server and SQL Server licenses
on Azure."
Key stat: Save up to 85% when combined with Reserved Instances.
Target audience: Customers migrating on-premises Windows/SQL workloads.

## Azure Pricing Calculator
Official page: https://azure.microsoft.com/pricing/calculator/
Official description: "Estimate the costs for Azure products and
services. Use the Pricing Calculator to estimate your monthly costs."
Use case: Pre-sales, architecture planning, budget approval.

## Azure Reservations
Official page: https://azure.microsoft.com/reservations/
Official description: "Save money by committing to one-year or
three-year plans for multiple products."
Key stat: Save up to 72% vs pay-as-you-go pricing.
Best for: Predictable, steady-state workloads.
Docs: https://learn.microsoft.com/azure/cost-management-billing/reservations/

## Azure Savings Plans for Compute
Official page: https://azure.microsoft.com/pricing/offers/savings-plan-compute/
Official description: "Azure savings plans for compute is a flexible
pricing model that provides savings up to 65 percent compared to
pay-as-you-go prices when you commit to spend a fixed hourly amount
on compute services for one or three years."
Key stat: Up to 65% savings. Applies to VMs, App Service,
Azure Container Instances, Azure Premium Functions.

## Azure Savings Plans for Databases
Docs: https://learn.microsoft.com/azure/cost-management-billing/savings-plan/
Covers: SQL Managed Instance, Azure Database for MySQL,
PostgreSQL, MariaDB.
Key stat: Save up to 55%.

## Azure Cost Management + Advisor
Official page: https://azure.microsoft.com/products/cost-management/
Official description: "Monitor, allocate, and optimize cloud costs
with transparency, accuracy, and efficiency."
Free tooling: budgets, alerts, anomaly detection, recommendations.
